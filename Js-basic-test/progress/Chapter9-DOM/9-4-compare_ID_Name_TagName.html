/* 区别getElementByID,getElementsByName,getElementsByTagName 以人来举例说明，人有能标识身份的身份证，有姓名，有类别(大人、小孩、老人)等。 1. ID 是一个人的身份证号码，是唯一的。所以通过getElementById获取的是指定的一个人。 2. Name 是他的名字，可以重复。所以通过getElementsByName获取名字相同的人集合。 3. TagName可看似某类，getElementsByTagName获取相同类的人集合。如获取小孩这类人，getElementsByTagName("小孩")。 把上面的例子转换到HTML中，如下: 音乐 input标签就像人的类别。 name属性就像人的姓名。 id属性就像人的身份证。 */
